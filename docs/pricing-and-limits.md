## 当前系统已支持的限额

以下限额可通过平台管理后台（SUPER_ADMIN）直接配置：

| 限额字段 | 对应租户字段 | 配置方式 | 说明 |
|---------|------------|---------|------|
| 知识库数量 | `max_knowledge_bases` | 平台管理 → 编辑租户 | NULL 表示不限 |
| 文档数量 | `max_documents` | 平台管理 → 编辑租户 | NULL 表示不限 |
| AI 日调用量 | `max_ai_daily_calls` | 平台管理 → 编辑租户 | NULL 表示不限 |
| 到期时间 | `expired_at` | 平台管理 → 编辑租户 | NULL 表示永不过期 |
| 门户标题 | `portal_title` | 租户管理 → 企业设置 | 租户自行配置 |
| 门户描述 | `portal_description` | 租户管理 → 企业设置 | 租户自行配置 |
| 主题色 | `theme_color` | 租户管理 → 企业设置 | 租户自行配置 |
| 门户启用 | `portal_enabled` | 租户管理 → 企业设置 | 租户自行开关 |

## 当前系统未实现（人工约定）

以下限额当前系统**不自动强制**，需在服务合同中约定，由平台管理员人工管理：

| 限额 | 说明 | 后续版本规划 |
|------|------|------------|
| 员工账号数量 | 当前不限制同一租户的员工数 | V0.4+ |
| 工单数量 | 当前不限制同一租户的工单数 | V0.4+ |
| 文件单次上传大小 | 当前硬编码 10MB | 可配置化 |
| API 调用频率（限流） | 当前无 IP/用户级限流 | V0.4 Redis 限流 |
| 数据导出 | 当前无导出功能 | V1.0 |
| 自定义域名 | 当前所有租户共用同一域名 | V1.0 |

## 定价策略建议

### 试用期

- 7 天免费试用，自动设置 `expired_at`
- 试用期满后门户显示"服务已到期"提示
- 试用期限额：知识库 1、文档 10、AI 日调用 50

### 升级流程

1. 客户确认套餐
2. 平台管理员通过管理后台修改租户限额和到期时间
3. 通过"重置密码"功能为客户生成新管理员密码（如需要）
4. 通知客户新的限额已生效

### 续费流程

1. 客户续费后，平台管理员延长 `expired_at`
2. 如套餐升级，同步调整 `max_*` 限额字段
3. 系统自动恢复正常使用
